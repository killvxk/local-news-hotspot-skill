## 功能特性

此技能可以：

1. **自动检测用户位置**
   - 通过 `curl ipinfo.io` 获取用户的地理位置
   - 提取城市、省份、国家信息

2. **采集多源新闻数据**
   - 基于 worldmonitor 项目分析的数据源结构
   - 优先使用 Tier 1（通讯社）和 Tier 2（主要媒体）来源
   - 包括：Reuters, AP, BBC, Bloomberg, Al Jazeera, CNN, Guardian 等

3. **智能分析和排名**
   - 时效性评分（0-40分）
   - 来源可信度评分（0-30分）
   - 本地相关性评分（0-30分）
   - 话题重要性评分（0-20分）

4. **本地化展示**
   - 显示 3-5 条全球热点新闻
   - 显示 3-5 条本地相关新闻
   - 终端友好的格式化输出
   - 使用表情符号增强可读性

## 触发方式

当用户说以下内容时会触发此技能：

## 技术实现

**数据源**: 使用 worldmonitor 项目验证过的 RSS URLs
- Tier 1: Reuters, AP, BBC, Bloomberg（通过 Google News 聚合）
- Tier 2: Al Jazeera, CNN, Guardian, NPR
- 区域源: 亚洲、欧洲、美洲、中东、非洲、中国特定来源
**聚合策略**: 主要使用 Google News RSS 聚合，提高可用性和稳定性
**位置检测**: ipinfo.io API
**输出格式**: 终端友好的文本格式
**缓存策略**: 15分钟缓存避免过度请求
**性能目标**: 30秒内完成
